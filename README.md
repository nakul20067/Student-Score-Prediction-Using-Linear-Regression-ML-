# Student Score Prediction Using Linear Regression

## Project Overview

This project uses Machine Learning to predict a student's exam score based on the number of hours studied. A Linear Regression model from Scikit-Learn is trained on historical data and evaluated using standard regression metrics.

## Objective

To understand and implement a simple supervised machine learning model for predicting student performance.

## Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn

## Dataset

The dataset contains:

* Hours Studied
* Score Obtained

## Workflow

1. Load dataset using Pandas
2. Split features and target variables
3. Train a Linear Regression model
4. Predict student scores
5. Evaluate model performance using:

   * Mean Absolute Error (MAE)
   * Mean Squared Error (MSE)
   * Root Mean Squared Error (RMSE)

## Example Prediction

Input: 7 Hours Studied

Output: Predicted Student Score

## Evaluation Metrics

* MAE
* MSE
* RMSE
